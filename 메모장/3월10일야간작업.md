# 3월 10일 야간 작업 요약

## 1. 작업 목적

- ROS2 기반 AK70 CAN 모터 드라이버 구조 정리
- 멀티쓰레딩 + command queue 구조 구현

## 2. 구현 구조

```text
/control_command
      ↓
subscriber callback
      ↓
latest command queue
      ↓
control thread (100Hz)
      ↓
AK70 MIT CAN command
      ↓
motor feedback
      ↓
/motor_state publish
```

## 3. 적용된 개념

### 멀티프로세스

- ROS2 노드 구조
- `ak70_can_node` / topic publisher

### 멀티쓰레딩

- ROS callback thread
- control loop thread
- state publish timer

### Queue

- `queue.Queue(maxsize=1)`
- 최신 명령만 유지하는 command buffer

## 4. 해결한 문제
